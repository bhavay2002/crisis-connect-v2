# 📊 2. RADAR CHART (SIGNAL FUSION)

## 🔷 Purpose

Show how different signals contribute:

```text
Text Signal      → 0.9
Location Risk    → 0.7
Repetition       → 0.6
User Trust       → 0.8
```

---

## 🔷 Implementation (Recharts)

```tsx
import { RadarChart, PolarGrid, PolarAngleAxis, Radar } from "recharts";

const data = [
  { signal: "Text", value: 0.9 },
  { signal: "Location", value: 0.7 },
  { signal: "Repetition", value: 0.6 },
  { signal: "Trust", value: 0.8 },
];

export function SignalRadar() {
  return (
    <RadarChart width={300} height={250} data={data}>
      <PolarGrid />
      <PolarAngleAxis dataKey="signal" />
      <Radar dataKey="value" stroke="#ef4444" fill="#ef4444" fillOpacity={0.6} />
    </RadarChart>
  );
}
```

# 📊 3. FACTOR BARS (WHY THIS DECISION?)

## 🔷 Purpose

Break down decision weight clearly:

```text
+0.8  keyword: "help"
+0.6  high-risk location
+0.4  repeated reports
-0.2  low user trust
```

---

## 🔷 Implementation

```tsx
export function FactorBars({ factors }) {
  return (
    <div className="space-y-2">
      {factors.map((f) => (
        <div key={f.label} className="flex items-center gap-2">
          <div className="w-32 text-sm">{f.label}</div>
          <div className="flex-1 bg-gray-700 h-2 rounded">
            <div
              className="bg-red-500 h-2 rounded"
              style={{ width: `${f.value * 100}%` }}
            />
          </div>
          <span className="text-xs">{f.value}</span>
        </div>
      ))}
    </div>
  );
}
```

---

## 🔷 UX Upgrade

* positive → green
* negative → red
* animate on load

# ⏱️ 4. DECISION TIMELINE (MOST POWERFUL)

## 🔷 Purpose

Show **how the decision evolved over time**

---

## 🔷 Example

```text
10:32 → User message received

10:33 → 3 similar reports detected
10:33 → Upgraded to HIGH
10:34 → Escalated to CRITICAL
```

---

## 🔷 Implementation

```tsx
function DecisionTimeline({ events }) {
  return (
    <div className="space-y-3">
      {events.map((e) => (
        <div key={e.id} className="flex gap-3">
          <div className="w-2 h-2 bg-red-500 rounded-full mt-2" />
          <div>
            <p className="text-sm">{e.message}</p>
            <span className="text-xs text-gray-400">{e.time}</span>
          </div>
        </div>
      ))}
    </div>
  );
}
```
