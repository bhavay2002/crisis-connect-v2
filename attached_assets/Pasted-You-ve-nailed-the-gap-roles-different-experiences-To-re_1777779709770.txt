## 🔷 Route-Level Separation (CRITICAL)

```text

  /user

  /admin
  /authority
```

👉 Each route = **independent layout + UX system**

---

## 🔷 Role Layout Wrapper

```tsx id="role_layout"
export function RoleLayout({ role, children }) {
  switch (role) {
    case "user":
      return <UserLayout>{children}</UserLayout>;
    case "volunteer":
      return <VolunteerLayout>{children}</VolunteerLayout>;
    case "admin":
      return <AdminLayout>{children}</AdminLayout>;
    case "authority":
      return <AuthorityLayout>{children}</AuthorityLayout>;
    default:
      return null;
  }
}
```
