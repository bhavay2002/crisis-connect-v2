## 🔷 Folder

```text
features/crisis/
  components/
    ActionPanel.tsx
    CrisisCard.tsx
  hooks/
    useCrisisRealtime.ts
  services/
    crisis.api.ts
  store/
    crisis.store.ts
  types/
    crisis.types.ts
```

---

## 🔷 API Layer (separate from UI)

```ts
// features/crisis/services/crisis.api.ts
import axios from "@/shared/services/api";

export const fetchCrises = () => axios.get("/api/reports");
export const dispatchResponder = (id: string) =>
  axios.post(`/api/crisis/${id}/dispatch`);
```

---

## 🔷 Store (local to feature)

```ts
// features/crisis/store/crisis.store.ts
import { create } from "zustand";

export const useCrisisStore = create((set) => ({
  crises: [],
  setCrises: (data) => set({ crises: data }),
}));
```

---

## 🔷 Hook (business logic lives here)

```ts
// features/crisis/hooks/useCrisisRealtime.ts
import { useEffect } from "react";
import { useCrisisStore } from "../store/crisis.store";

export function useCrisisRealtime(socket) {
  const setCrises = useCrisisStore((s) => s.setCrises);

  useEffect(() => {
    socket.on("CRISIS_UPDATE", setCrises);
    return () => socket.off("CRISIS_UPDATE");
  }, [socket, setCrises]);
}
```

---

## 🔷 Component (UI only)

```tsx
// features/crisis/components/CrisisCard.tsx
export function CrisisCard({ crisis }) {
  return (
    <div className="card">
      <h3>{crisis.title}</h3>
      <p>{crisis.location}</p>
    </div>
  );
}
```

---

👉 Notice:

* ❌ no API calls in component
* ❌ no business logic in UI
* ✅ clean separation

## 3. “Single Responsibility per File”
